# üåç Flag Flashcards

An interactive flashcard app to help you learn country flags and capitals around the world!

## Features

- **196 Country Flags**: Learn flags from all recognized countries across all continents
- **High-Quality SVG Flags**: Beautiful, scalable flag images from the flag-icons library
- **Comprehensive Data**: Includes country name, capital, region, subregion, and population
- **Interactive Flashcards**: Click or press spacebar to flip cards and reveal answers
- **Score Tracking**: Keep track of which flags you know and don't know
- **Keyboard Navigation**: Use arrow keys to navigate, spacebar to flip
- **Shuffle Mode**: Randomize the deck to test your knowledge
- **Modern Design**: Gradient backgrounds, smooth animations, and polished UI
- **Responsive Design**: Works beautifully on desktop and mobile devices

## How to Use

1. **Open the App**: Simply open `index.html` in your web browser
2. **View the Flag**: Each card shows a country's flag in high-quality SVG
3. **Flip the Card**: Click the card or press `Space`/`Enter` to reveal the country name, capital, region, and population
4. **Rate Yourself**: 
   - Click "‚úì I Know This" (or press `1`) if you knew the answer
   - Click "‚úó Don't Know" (or press `2`) if you didn't know
5. **Navigate**: Use "Previous" and "Next" buttons or arrow keys to move between cards
6. **Shuffle**: Click "üîÄ Shuffle" to randomize the deck and reset your score

## Running Locally

No build process required! Just open `index.html` in any modern web browser:

```bash
# Option 1: Open directly
open index.html

# Option 2: Use a local server (recommended)
python3 -m http.server 8000
# Then visit http://localhost:8000
```
